## Contexto

Repositorio del curso mobile web de Platzi. En este punto el proyecto está **pausado**: el `index.html` solo tiene la estructura base (`<header>`, `<main>` con cuatro `<section>` vacíos y `<footer>`), la tipografía importada desde Google Fonts (DM Sans e Inter) y un `style.css` que apenas contiene el comentario con el orden de propiedades sugerido por Platzi (Posicionamiento, Box model, Tipografía, Visuales, Otros).

Los assets SVG y PNG ya están cargados en la carpeta `assets/` pero todavía no se usan en el HTML.

## Tecnologías

- HTML5
- CSS3
- Google Fonts — DM Sans, Inter
- Assets SVG/PNG

## Cómo abrir

No hay build ni dependencias. Abrir `index.html` directamente en el navegador (doble click) o servirlo con **Live Server** en VS Code.

## Estructura

```
CursoMobile/
├── index.html     # esqueleto con header, 4 sections, footer (sin contenido)
├── style.css      # solo un comentario guía, sin reglas aún
├── assets/        # 11 SVG + 2 PNG listos para usar
└── README.md
```

## Estado

Proyecto pausado. El esqueleto y los assets están listos, pero las secciones aún no tienen contenido maquetado.
